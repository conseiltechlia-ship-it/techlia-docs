# Techlia — Documentation

Ce dépôt centralise la documentation stratégique et technique de Techlia.

## Structure

- `docs/` — documents de fond (vision, positionnement, personas, architecture, SEO...)
- `assets/` — logo, icônes, captures d'écran, illustrations
- `prompts/` — prompts réutilisables pour Claude, Lovable, Cursor
- `roadmap/` — roadmap et backlog
- `branding/` — charte graphique et tone of voice
