# Techlia — Documentation

Ce dépôt centralise la documentation stratégique et technique de Techlia.

## Structure

- `docs/` — documents de fond (vision, positionnement, personas, architecture, SEO...)
- `prompts/` — prompts réutilisables (Claude, Lovable, Cursor, ChatGPT)
- `roadmap/` — roadmap, backlog, releases, changelog
- `assets/` — logo, icônes, captures d'écran, illustrations, mockups, fonts
- `dashboard/` — spécifications des tableaux de bord clients
- `sales/` — supports commerciaux (offres, pricing, objections, propositions)
- `seo/` — stratégie mots-clés, villes, secteurs, maillage interne
- `presentation/` — scripts de démo, pitch, checklist réunion
